# BEABÁ – Sistema de Gestão Escolar

## 📘 Sobre o Projeto
**BEABÁ** é um sistema de gestão escolar desenvolvido para atender escolas da organização
**Operação Mato Grosso**, com foco na Educação Básica.

O sistema foi projetado para:
- Atender BNCC e Censo Escolar (INEP)
- Garantir histórico escolar imutável
- Evitar retrabalho administrativo
- Ser auditável e escalável

## 🏫 Público-alvo
Escolas da **Operação Mato Grosso**, em contextos urbanos e rurais.
