# BEABÁ – Sistema de Gestão Escolar

## Apresentação
BEABÁ é um sistema livre de gestão escolar desenvolvido para as escolas da Operação Mato Grosso,
voltado à Educação Básica, alinhado à BNCC e ao Censo Escolar (INEP).
